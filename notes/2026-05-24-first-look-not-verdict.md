# first look is not a verdict

A quick link check can answer small questions:

- where did the URL end up?
- what title did the page show?
- did it redirect somewhere weird?
- did the page ask for too much?

It cannot prove a site is safe.
It cannot prove a project is legit.
It cannot replace reading docs, checking contracts, or using a clean browser profile.

Good first-look notes say what was checked and what stayed unknown.

Example:

```text
checked:
- final URL
- page title
- obvious form fields

not checked:
- contract code
- wallet interaction
- team identity
- download contents
```

Small checks help.
Fake certainty does not.
