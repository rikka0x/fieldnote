# 2026-05-22 why links lie

Links lie by looking normal.

Not always on purpose. Sometimes teams move domains, marketing tools wrap URLs, and docs rot quietly. Still: bad links and sloppy links feel the same when a wallet popup appears.

Small habit:

- read the domain slowly
- check official sources before clicking campaign links
- treat weird subdomains like tiny alarms
- avoid urgency bait
- never paste secrets into forms
- stop when a page asks for wallet approval before explaining itself

A good project can survive one more manual check.
